 # StudyFlow Alpha


A study session tracker built in F# using the Fable compiler and the Elmish (Model-View-Update) architecture. The app runs entirely in the browser — F# compiles to JavaScript via Fable, bundled with webpack.
This project extends my earlier StudyFlow dashboard by rebuilding it from scratch in a functional language, with a proper unidirectional data flow and immutable state model instead of the original vanilla JS approach.
Features: - Track study sessions by subject with duration and notes
          - Dashboard with total study time and per-subject progress bars
          - Session history with the ability to delete entries
          - Add custom subjects on the fly 
Tech Stack: - F# with Fable (F# to JavaScript compiler)
            - Elmish for the Model-View-Update architecture
            - Fable.React for the UI
            - Webpack for bundling
            - GitHub Actions for CI/CD deployment to GitHub Pages
